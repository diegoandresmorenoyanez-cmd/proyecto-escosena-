# 📋 Requisitos del Sistema - Portal Informativo

## ✅ Requisitos Funcionales
- **RF01:** El sistema debe permitir el registro de usuarios (Aprendices, Instructores, Administrativos).
- **RF02:** El sistema debe permitir iniciar sesión mediante un portal seguro.
- **RF03:** El sistema debe permitir a los administradores e instructores crear, editar y eliminar noticias o comunicados oficiales.
- **RF04:** El sistema debe permitir filtrar las noticias por categorías (Académico, Bienestar, Tecnología, Convocatorias).

---

## ⚙️ Requisitos No Funcionales
- **RNF01:** La interfaz debe ser responsive (adaptable a celulares y computadores).
- **RNF02:** El tiempo de respuesta del portal no debe superar los 3 segundos.
- **RNF03:** Las contraseñas deben estar encriptadas en la base de datos por seguridad.
